# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added
- Initial release of AutoPromo Video.
- Automated web capture using Puppeteer.
- Cinematic camera movements with Pro Director 2.0.
- Support for 16:9 and 9:16 aspect ratios.
- Background music integration.
- Custom logo overlay support.
- Queue management with Redis.
- Cloud storage integration (Cloudflare R2 / AWS S3).
- REST API for programmatic video generation.
- Professional documentation suite (API, Config, Deployment, etc.).
- GitHub issue and pull request templates.
